# lnbits-rs

A comprehensive Rust client for the [LNbits](https://lnbits.com/) API. LNbits is a free and open-source lightning wallet and account system.

## Key features

- [x] Create invoices
- [x] Webhook for incoming invoices
- [x] Decode invoices
- [x] Pay invoices
- [x] Get wallet details
- [x] Tor support
